# B14-A01-ASSIGNMENT-01

A modern and responsive landing page for **DevConf 2026**, built using HTML and CSS. The project showcases a conference website with a clean layout, reusable components, and responsive design.

## Features

| Feature | Description |
|---------|-------------|
| Responsive Navbar | Logo, navigation links, and call-to-action button |
| Hero Section | Full-width banner with conference information |
| Speakers | 2×2 speaker card layout |
| Pricing | Three pricing plans with a featured Pro plan |
| Conference Highlights | Additional AI-assisted section showcasing event statistics |
| Footer | Logo, social media links, and copyright information |

## Technologies Used

| Technology | Purpose |
|------------|---------|
| HTML5 | Page structure |
| CSS3 | Styling and layout |
| Flexbox | Navigation and footer layout |
| CSS Grid | Speaker and highlight sections |
| CSS Animations | Hover effects and entrance animations |
